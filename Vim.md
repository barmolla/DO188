# Documentación VIm

## Subrayar línea en la que está ubicada el cursor

shift + :

```
set cursorline
```

## Visualizar número de línea a la izquierda

shift + :
```
set number
```

## Iluminar columna en la que está ubicada el cursor

shift + :
```
set cursorcolumn
```

## Habilitar sintáxis de diversos lenguajes de scripting

shift + :
```
syntax on
```
